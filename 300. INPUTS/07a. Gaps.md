# Podstawowa składnia SUMX

Które funkcje poprawnie uzupełniają składnię funkcji iteracyjnej obliczającej całkowitą wartość sprzedaży?

## Dostępne funkcje

- SUM
- SUMX
- CALCULATE
- FILTER

## Kod do uzupełnienia

```dax
Wartosc Sprzedazy = 
    [SLOT:0](
        Sprzedaz,
        Sprzedaz[Cena] * Sprzedaz[Ilosc]
    )
```

## Poprawne rozwiązanie

SUMX

## Feedback poprawny

✅Brawo! `SUMX` to funkcja iteracyjna, która przechodzi przez każdy wiersz tabeli Sprzedaż, mnoży Cenę przez Ilość dla każdego wiersza osobno, a następnie sumuje wszystkie wyniki.

## Wskazówka

Funkcje iteracyjne w DAX kończą się literą `X`. Szukaj funkcji, która najpierw mnoży wartości w każdym wierszu, a potem je sumuje.

# Obliczanie marży z AVERAGEX

Która funkcja pozwoli obliczyć średnią marżę dla każdego wiersza transakcji?

## Dostępne funkcje

- AVERAGE
- AVERAGEX
- SUMX
- CALCULATE

## Kod do uzupełnienia

```dax
Srednia Marza = 
    [SLOT:0](
        Sprzedaz,
        Sprzedaz[Wartosc Netto] - Sprzedaz[Koszty Netto]
    )
```

## Poprawne rozwiązanie

AVERAGEX

## Feedback poprawny

✅Brawo! `AVERAGEX` iteruje przez każdy wiersz tabeli Sprzedaż, oblicza marżę dla każdej transakcji odejmując Koszty od Wartości Netto, a następnie oblicza średnią z wszystkich wyników.

## Wskazówka

Potrzebujesz funkcji iteracyjnej, która obliczy różnicę dla każdego wiersza, a następnie zwróci średnią arytmetyczną tych wartości.

# Warunkowe liczenie z COUNTX

Która funkcja pozwoli policzyć zamówienia powyżej 1000 PLN wykonując sprawdzenie dla każdego wiersza?

## Dostępne funkcje

- COUNT
- COUNTX
- Sprzedaz
- COUNTROWS

## Kod do uzupełnienia

```dax
Liczba Duzych Zamowien = 
    [SLOT:0](
        [SLOT:1],
        IF(Sprzedaz[Wartosc Netto] > 1000, 1, BLANK())
    )
```

## Poprawne rozwiązanie

COUNTX,Sprzedaz

## Feedback poprawny

✅Brawo! `COUNTX` iteruje przez każdy wiersz tabeli Sprzedaż, sprawdza warunek dla każdego wiersza osobno i liczy ile razy wyrażenie zwróciło niepustą wartość. Gdy Wartość Netto przekracza 1000, funkcja IF zwraca 1, które jest liczone.

## Wskazówka

Potrzebujesz funkcji iteracyjnej, która liczy ile razy wyrażenie zwróciło niepustą wartość. Funkcja ta kończy się literą `X` i zaczyna od `COUNT`.
