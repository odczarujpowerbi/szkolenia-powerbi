Sprzedaż Komputerów

Napisz miarę sumującą sprzedaż dla produktu **Komputer** używając najlepszych praktyk DAX. Miara nie powinna brać pod uwagę innych produktów.
CALCULATE FILTER SUM SUMX
Sprzedaż Komputerów =
(
(Sprzedaż[Kwota]),
Produkty[Nazwa] = 'Komputer'
)
Najlepszą praktyką DAX jest stosowanie funkcji **CALCULATE** do modyfikowania kontekstu filtru wraz z prostym filtrowaniem, zamiast używania **FILTER**, które tworzy dodatkową tabelę w pamięci i może obniżyć wydajność.

Total Sprzedaży

Utwórz miarę obliczającą całkowitą sprzedaż dla wszystkich produktów, ignorując wszelkie filtry na tabeli Produkty.
CALCULATE ALL SUM SUMX
Total Sprzedaży =
(
(Sprzedaż[Kwota]),
(Produkty)
)
Użyj funkcji ALL aby usunąć filtry z tabeli i obliczyć wartość dla całego zbioru danych.

Ranking produktów

Stwórz miarę obliczającą pozycję (ranking) produktu względem sprzedaży. Im wyższa sprzedaż, tym niższa pozycja (1 = najlepsza).
RANKX ALL CALCULATE SUM
Ranking Produktu =
(
(Produkty),
([Sprzedaż])
)
RANKX wymaga tabeli do iteracji i wyrażenia zwracającego wartość do porównania. Użyj ALL aby uzyskać wszystkie produkty bez filtrów.
